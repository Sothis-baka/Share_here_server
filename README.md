# Share here

### Author

sothis-baka

### Description

An online forum.

Using Apollo-server, Graphql as backend, React as front end.

Using Bootstrap. Grid for layout and icons.

### Use

run 

```
npm start
```

in each folder

### Deployment

Client deployed at [Heroku](https://share-here-cli.herokuapp.com/)
